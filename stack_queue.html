Stack Operations

Stack is a linear data structure which follows the LIFO (Last In First Out) principle. Common operations include push, pop, peek and checking if the stack is empty.

Queue Operations

Queue is a linear data structure which follows the FIFO (First In First Out) principle. Common operations include enqueue, dequeue, front, and checking if the queue is empty.

Types of Queue

1. Simple Queue: Elements are inserted at rear and deleted from front.

2. Circular Queue: Rear connects to front to make a circle.

3. Priority Queue: Each element has a priority and highest priority elements are dequeued first.
